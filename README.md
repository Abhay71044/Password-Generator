# 🔐 Password Generator

A responsive and secure Password Generator built using HTML, CSS, and JavaScript.

## 🚀 Features

- Generate strong random passwords
- Adjustable password length using slider
- Include:
  - Uppercase Letters (A-Z)
  - Lowercase Letters (a-z)
  - Numbers (0-9)
  - Special Symbols (!@#$%^&*)
- Password strength indicator
- One-click copy to clipboard
- Responsive UI
- Randomized password generation using Fisher-Yates Shuffle

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
